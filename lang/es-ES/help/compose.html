Ayuda de Enigmail

Uso de Enigmail para redactar mensajes

  • Menú Enigmail en la ventana de redacción de mensajes
    • Firmar mensaje: Activa o desactiva el envío de correo firmado. El usuario recibe una notificacón si hay un fallo en la firma.
    • Cifrar mensaje: Activa o desactiva el cifrado para todos los destinatarios antes de enviarlo. El usuario recibe una notificacón si hay un fallo al cifrar.

        Si está activada la opción Mostrar la selección si es necesario en Preferencias -> pestaña Selección de clave, se abrirá una ventana con una lista de claves si hay direcciones en la lista de destinatarios del mensaje de las que no se tiene la clave pública.

        Si está activada la opción Nunca mostrar el cuadro de selección de clave OpenPGP en Preferencias -> pestaña Selección de clave, y hay direcciones en la lista de destinatarios del mensaje de las que no se tiene la clave pública, el mensaje se enviará sin cifrar.

    • Usar PGP/MIME en este mensaje: Activa o desactiva el uso de PGP/MIME en este mensaje.

        Si sabe con certeza que los destinatarios pueden leer el correo usando el formato PGP/MIME, entonces se debe usar.

        Esta característica depende de la configuración en Preferencias -> pestaña PGP/MIME si está establecida en Permitir usar PGP/MIME o Usar siempre PGP/MIME.

    • Submenú Opciones predeterminadas de redacción:
      • Opciones de firmado/cifrado...: acceso directo a Configuración de la cuenta-> Opciones OpenPGP, en el cliente de correo.
      • Opciones de envío...: acceso directo a la pestaña Preferencias -> Enviar.
      • Opciones de selección de clave...: acceso directo a la pestañaPreferencias -> Selección de clave.
      • Opciones PGP/MIME...: acceso directo a la pestaña Preferencias -> PGP/MIME.
    • Deshacer cifrado: Si hay un fallo al enviar el correo, como un fallo en el servidor POP que no acepta la solicitud, Enigmail no lo sabrá, y el mensaje cifrado seguirá mostrándose en la ventana de redacción. Con esta opción de menú, se deshará el cifrado o firma, volviendo a la ventana de redacción con su texto original.
      Como solución temporal, también se puede usar para descifrar el texto citado al contestar a mensajes cifrados. Enigmail debe descifrar automáticamente el mensaje citado, pero si se produce un fallo por cualquier motivo, se puede usar esta opción del menu para forzarlo.
    • Insertar clave pública: inserta el bloque de la clave en ASCII-armored en la ubicación actual del cursor en la ventana de redacción. Se le preguntará por las direcciones de correo de las claves que se van a insertar. Las claves insertadas de esta forma, se reconocerán automáticamente en la recepción por Enigmail. Después de insertar la clave, todavía puede elegir si firmar o cifrar el correo según se necesite. Tampoco inserte más de un bloque de en un mensaje; simplemente especique varias direcciones de correo, separadas por comas o espacios, cuando se le pregunte.
    • Limpiar clave salvada: Limpia las contraseñas introducidas manualmente de la caché. Esto es útil si se tienen varias contraseñas.
    • Ayuda: Muestra información de ayuda desde el sitio web.
