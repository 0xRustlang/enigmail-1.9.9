Información de actualización para Enigmail 0.8x

¡Enigmail v0.8x para Thunderbird ha recibido una nueva interfaz de usuario! Si está familiarizado con versiones de Enigmail anteriores a la 0.81.0, por favor, lea esta información antes de proceder.

Configurar Enigmail

La configuación de Enigmail ahora está separada en dos partes:

  • Opciones específicas de la cuenta para el ID de la clave PGP, cifrado / firmado predeterminados
  • Preferencias globales comunes

Opciones específicas de la cuenta

Las opciones específicas de la cuenta están disponibles a través del menúHerramientas -> Configuración de la cuenta. Para cada cuenta ahora hay una lista de elementos de Seguridad OpenPGP que mantiene las opciones específicas para cada cuenta.

La configuración de la clave PGP desde Enigmail v0.80.0 y versiones anteriores no están convertidas a la nueva configuración por cuenta. ¡Enigmail está   desactivado por defecto para todas las cuentas!

Preferencias globales comunes

La configuración global comun se ha reconstruido y ahora está en un único cuadro de diálogo Las preferencias están disponibles a través del menú de Enigmail o pulsando el botón Avanzadas desde la configuración de la cuenta en el cliente, y luego Seguridad OpenPGP.

Se ha añadido una nueva opción que puede activar el cifrado automáticamente si se responde a un mensaje cifrad. Hay que tener en cuenta que esto no influye en la configuración de firma de mensajes.

Redacción de mensajes

El comportamiento de Enigmail se ha adaptado para que sea similar al de S/MIME. Se ha añadido un botón nuevo OpenPGP desde donde se puede seleccionar la configuración para cifrar o firmar. El menú Enigmail ya no sirve para enviar correo cifrado o firmado, ya que esto se hace únicamente desde el botón normal de Enviar o el menú Archivo -> Enviar Se recomienda que se prueben las opciones nuevas mandando un par de mensajes a uno mismo antes de usarlo activamente, para comprender el nuevo comportamiento.

Las opciones para cifrar o firmar un mensaje están ahora visibles en la esquina inferior derecha de la ventana de redacción de mensajes, con iconos pequeños que se muestran si firmar o cifrar está activado.

Problemas conocidos

  • Enigmail todavía no es capaz de recordar varias contraseñas para distintas claves PGP O bien se escribe la contraseña cada vez, o se pone la misma contraseña para todas las claves que se usen.
