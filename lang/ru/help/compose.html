Применение Enigmail при создании сообщения

Меню Enigmail в окне создания сообщения
Подписать сообщение
Включить/выключить подпись при отправке сообщения. Пользователь оповещается, если подписать не удается.
Шифровать сообщение

Включить/выключить при отправке шифрование для всех адресатов. Пользователь оповещается, если зашифровать не удается.

Если в меню Настройки -> Подбор ключа установлен флаг При необходимости показывать варианты, то в случаях, когда для каких-то адресов в списке получателей у вас нет открытых ключей, будет выводиться список ключей.

Если в меню Настройки -> Подбор ключа установлен флаг Никогда не показывать окно выбора ключей и для каких-то адресов в списке получателей у вас нет открытых ключей, сообщение будет отправлено незашифрованным.

Применить для этого сообщения PGP/MIME
Включить/выключить PGP/MIME для этого сообщения.

Если вы знаете, что адресаты могут читать почту в этом формате, то им нужно пользоваться.

Эта функция требует, чтобы в меню Настройки -> PGP/MIME было установлено Разрешить применение PGP/MIME или Всегда применять PGP/MIME.

Исходные параметры
  • Параметры шифрования/подписи...: сокращение Параметры учетной записи -> Параметры OpenPGP.
  • Параметры отправки...: сокращение Настройки -> Отправка.
  • Параметры подбора ключей...: сокращение Настройки -> Подбор ключей.
  • Параметры PGP/MIME... Настройки -> PGP/MIME.
Отменить шифрование

Когда при собственно отправке сообщения происходит сбой, например, сервер POP не принимает запрос, в Enigmail информация об этом не поступает и зашифрованное сообщение по-прежнему отображается в окне создания сообщения. Этот пункт меню отменяет шифрование/подпись, возвращая в окно сообщения первоначальный текст.

В качестве временной меры этот пункт можно применять также для расшифрования зашифрованного текста при ответе на зашифрованные сообщения. Цитируемое сообщение должно расшифровываться автоматически, но если по какой-то причине этого не происходит, то этим пунктом меню можно воспользоваться для принудительного расшифрования.

Вставить открытый ключ
Вставить блок с открытыми ключами в оболочке ASCII в текущей позиции курсора в окне создаваемого сообщения. Программа запросит адреса электронной почты для вставляемых ключей. Вставленные таким образом ключи автоматически распознаются Enigmail на принимающей стороне. После вставки ключей сообщение можно шифровать и подписывать по мере необходимости. Не вставляйте несколько блоков в сообщение; вместо этого укажите несколько адресов, разделенных пробелами или запятыми.
