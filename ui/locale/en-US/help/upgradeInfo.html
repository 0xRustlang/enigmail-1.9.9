What's New in Enigmail v2.0?

Welcome to Enigmail v2.0! We introduced a lot of new features:

Hiding the Message Subject

We developed a new method that hides the email subject by replacing the visible subject with "Encrypted Mesage". Once such a message is decrypted, the original subject is replaced automatically. (Note: this feature requires the message to be sent with PGP/MIME.)

Improved behavior of Encrypt and Sign Buttons

The Encrpyt and Sign buttons in the message composer window now work for both the OpenPGP and S/MIME protocols. If both algorithms are possible, then Enigmail will try to prefer the one for which all keys are available.

Support for Autocrypt

Enigmail now supports Autocrypt, a new standard to distribute keys as part of sent messages. Enigmail automatically imports keys from Autocrypt-compliant messages, such that over time more and more emails can be encrypted.

Support for p≡p (Pretty Easy Privacy)

Enigmail now contains a p≡p Junior Mode. You currently need to manually install p≡p for this purpose; this will change in a future release. The p≡p Junior Mode allows you to use OpenPGP encryption as transparently as possible; you don't need to care for key management and synchronization of keys between devices anymore.
